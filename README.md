# Student CMS

A College Management System built with Node.js, Express.js, MongoDB, and EJS.

## Features

- Admin Dashboard
- Staff Dashboard
- Student Dashboard
- Attendance Management
- Course Management
- Exam Scores
- Leave Management
- Notifications
